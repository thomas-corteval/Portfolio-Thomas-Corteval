SAE 01.04 : Elaboration et création d’une base de donnée – Analyse des besoins du client

Dans le cadre de mon premier semestre universitaire, ce projet à réaliser en équipe de 4 personnes qui incarnait une équipe d’administration base de données dans une grosse agence WEB. Où un nouveau projet vient d’être lancé, l’étude du besoin a été faite et nous allons devoir créer une base de données adaptée au projet à partir de celui-ci. Ce projet est un outil WEB afin de pouvoir organiser un game jam. Cet outil doit permettre de référencer les participants, les groupes, le classement et les notes finales, ainsi que les jeux, afin d’avoir des archives de chaque édition pour pouvoir faire des rétrospectives.

Différente étape du projet :

  • Études des exigences et des besoin du client.
  • Création d’un MCD.
  • Création d’un MLD.
  • Créer la base de donnée avec nos requêtes.
  • Implémenter un jeu de donnée afin d’exploiter et de tester la base de données.
  • Créer un jeu de requêtes afin de répondre à toutes les exigences du client sur son outil web.

Mission que j’ai réalisé pendant ce projet :

  • Étude des exigences et des besoins du client.
  • Élaboration d’un MCD avec mon équipe.
  • Création d’une partie des requêtes de création de table de notre base de donnée.
  • Création d’une partie des requêtes sur la base de données permettant de répondre aux exigences du client.

L’ensemble des compétence que j’ai dû mobiliser durant ce projet :

  • Analyse des besoins du client
    • En réussissant à identifier et à comprendre les besoins spécifiques du client en matière de base de données.
  • Conception de modèles de données
    • Grâce à l’élaboration d’un MCD, décrivant les entités, les relations et les contraintes nécessaires pour représenter efficacement les données du projet.
  • Création de requêtes SQL
    • En créant des requêtes SQL pour la création de tables et pour répondre aux besoins spécifiques du projet.
  • Compréhension des bases de données relationnelles
    • En appliquant les concepts de bases de données relationnelles et de leur application pratique dans un projet réel.
  • Travail d'équipe
    • En travaillant en collaboration avec mes camarades afin de mener ce projet à bien.
