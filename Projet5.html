SAE 02.01 : Développement d’un système de visualisation de données

Dans le cadre de mon second semestre universitaire, ce projet à réaliser en équipe de 2 personnes avait pour but d’essayer de constater le phénomène de réchauffement climatique à partir des données. Afin d’exploiter au mieux ces données qui sont considérables en quantité, une étude sur la représentation de plusieurs types de structures de données fut nécessaire. L’objectif était de déterminer quelle structure semble être la plus adaptée pour pouvoir étudier au mieux le réchauffement climatique.

Différente étape du projet :

  • Récupérer et trier des données conséquentes d’un tableur.
  • Afficher ces données sous forme de graphiques
    • Créer une class pour gérer les différents graphiques
    • Créer une class afin d’afficher les différents graphiques
  • Choix d'arborescence

Mission que j’ai réalisé pendant ce projet :

  • Coder des parties d’algorythme.
  • Réalisation d’un graphique de données dans directement dans visual studio grâce à une bibliothèque graphique.

L’ensemble des compétence que j’ai dû mobiliser durant ce projet :

  • Programmation en C++
    • Pour coder les différentes parties de l’algorithme.
  • Structures de données
    • Conception de différentes classes pour créer et afficher des graphiques.
  • Manipulation de données
    • Travaille sur des données conséquentes, ce qui a nécessité des compétences en lecture, en extraction et en traitement des données.
  • Travail d'équipe
    • En travaillant en collaboration avec mes camarades afin de mener ce projet à bien.
